# Sales Performance Dashboard

## Project Overview
This project analyzes sales data using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI

## Features
- Total Sales Analysis
- Total Profit Analysis
- Top Selling Product Identification
- Product-wise Sales Comparison
- Interactive Dashboard Filters
- Sales Data Visualization

## Files
- sales_data.csv : Dataset
- sales_analysis.py : Sales and Profit Analysis
- chart.py : Sales Chart Generation
- sales_chart.png : Generated Chart
- Sales_Dashboard.pbix : Power BI Dashboard

## Results
- Total Sales: 299,000
- Total Profit: 70,000
- Top Selling Product: Laptop
