# 🛒 E-Commerce Sales Dashboard

## 📌 Project Overview
This project analyzes e-commerce sales data using Python and Power BI. The dashboard provides insights into revenue, profit, orders, sales by category, and sales by region.

## 🛠️ Tools Used
- Python
- Pandas
- Power BI
- CSV Dataset

## 📊 Features
- Total Revenue Analysis
- Total Profit Analysis
- Total Orders Count
- Sales by Category
- Sales by Region
- Interactive Filters (Region & Category)
- Orders Details Table

## 📂 Project Files

- ecommerce_analysis.py
- ecommerce_data.csv
- Ecommerce_Sales_Dashboard.pbix
- dashboard_screenshot.png

## 📈 Dashboard KPIs

- Total Revenue: 267K
- Total Profit: 60K
- Total Orders: 10

## 🚀 How to Run

### Python Analysis
```bash
python ecommerce_analysis.py
```

### Power BI Dashboard
1. Open `Ecommerce_Sales_Dashboard.pbix`
2. Refresh the dataset if required.
3. Use Region and Category filters for analysis.
